My Coding Journey

First Steps: My programming journey began with HTML and CSS, where I learned to structure web pages and make them visually appealing. Building simple static websites gave me my first taste of creating something digital.

Becoming Interactive: JavaScript opened a new world of possibilities, allowing me to add interactivity to my projects. I created my first form validations, image sliders, and simple games, which boosted my confidence significantly.

Expanding Horizons: Venturing into Python and C++ helped me understand core programming concepts like data structures and algorithms. I particularly enjoyed solving logic puzzles and building small applications.

Current Focus: Now I'm exploring full-stack development, learning how to build complete web applications. Each day brings new challenges and learning opportunities as I work toward becoming a professional developer.
